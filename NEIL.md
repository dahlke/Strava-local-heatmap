# My Commands

Here is a [https://boundingbox.klokantech.com/](tool) to get the bounding box very easily.

## Bay Area 2020
python strava_local_heatmap.py --year 2020  --zoom 12 --bounds 37.629481 37.973602 -122.656086 -122.188748 --output bay_area_2020.png

## San Francisco 2020
python strava_local_heatmap.py --year 2020  --zoom 14 --bounds 37.699908 37.816528 -122.528357 -122.35146 --output san_francisco_2020.png

## Bay Area All Time
python strava_local_heatmap.py --zoom 12 --bounds 37.629481 37.973602 -122.656086 -122.188748 --output bay_area_all_time.png

## San Francisco All Time
python strava_local_heatmap.py --zoom 14 --bounds 37.699908 37.816528 -122.528357 -122.35146 --output san_francisco_all_time.png
